# Syncelle – Landing Page

Landing page para la startup tecnológica **Syncelle**.

Syncelle ofrece páginas web estáticas bajo un modelo de suscripción mensual: el cliente paga por el tiempo que quiere tener su página activa. Si deja de pagar, su página se da de baja automáticamente. Propuesta simple, profesional y directa: presencia digital elegante sin complicaciones.

## Estructura del proyecto

- `index.html`: Página principal
- `styles.css`: Estilos principales (responsive, minimalista, moderno)
- `script.js`: Lógica JS para el formulario de contacto
- `logo.png`: Logo de la marca (usado en el hero)

## Paleta y tipografía
- Fondo: `#0B0B0B`
- Texto: `#FFFFFF`
- Verde acento: `#00F4AE`
- Verde secundario: `#00DA8E`
- Tipografía: Inter o Poppins (Google Fonts)

## Despliegue

Puedes desplegar este sitio en **Netlify** o **GitHub Pages** fácilmente:

### Netlify
1. Sube el repositorio a GitHub.
2. Ve a [netlify.com](https://netlify.com) y conecta tu repo.
3. Netlify detecta automáticamente el `index.html` y publica el sitio.

### GitHub Pages
1. Sube el repositorio a GitHub.
2. Ve a Settings > Pages.
3. Selecciona la rama principal y la carpeta raíz (`/`).
4. ¡Listo! El sitio estará disponible en tu URL de GitHub Pages.
